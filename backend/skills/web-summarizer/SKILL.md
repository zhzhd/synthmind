# Web Summarizer Skill

When the user asks you to summarize a webpage or article, use this
skill to fetch the content and provide a concise summary.

## Instructions

1. First, confirm the URL with the user if unclear.
2. Use `web_search` to find the page if only a topic is given.
3. Read the content and provide a summary with:
   - Main topic / thesis
   - Key points (3-5 bullet points)
   - Notable quotes or data
   - Source attribution
4. Keep the summary concise but comprehensive.
